## 基本命令

所有命令都在项目根目录运行：

| 基本命令                 | 操作                                   |
| :----------------------- | :------------------------------------- |
| `pnpm install`           | 安装依赖                               |
| `pnpm dev`               | 在 `localhost:4321` 启动本地开发服务器 |
| `pnpm build`             | 构建生产站点到 `./dist/`               |
| `pnpm preview`           | 在部署前本地预览构建                   |
| `pnpm check`             | 运行 Astro 错误检查                    |
| `pnpm format`            | 使用 Prettier 格式化代码               |
| `pnpm lint`              | 检查并修复代码问题                     |
| `pnpm astro ...`         | 运行 Astro CLI 命令                  |
